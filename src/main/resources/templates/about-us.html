" WEB BASED BLOOD DONATION MANAGEMENT SYSTEM"

The main goal of the blood donation management system project is to monitor blood bank data,blood cell,blood stock,donar and receiver directory

The projects aim is to deevelop an application system to minimize the manual work for blood management.

Our project will help the medical center to collect blood from donor and store blood in blood bank.

Languages used:
- HTML
- CSS
- bootstrap
- Java advance
Applications used:
- spring tool suit
- Tomcat server 9.0
- Heroku (PAAS Cloud)
